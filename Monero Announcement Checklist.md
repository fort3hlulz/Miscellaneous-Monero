# Monero Announcement Checklist
This checklist serves as a simple guide to make sure that no announcement channel falls behind when a new update, blog, post, etc are released.  

# Checklist
- [ ] Delta Direct
- [ ] Blockfolio Signal
- [ ] CoinGecko Beam
- [ ] @monero Twitter
- [ ] r/Monero
- [ ] monero-announce mailer (optional)
